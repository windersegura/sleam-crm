# CRM app. This app allows user to track their inventory and sells. It uses react and TanStack Router. The app is built with TanStack Router. Also it calls a Rest API with Express

We will using npm

## Features
* Loggin users
* Create inventory
* Create product
* Edit product
* Delete product
* View product
* List product
* Add product to inventory

## Instalation
```bash
npm install
```
